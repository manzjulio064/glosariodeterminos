Regla de 3

ChatGPT Plus

La regla de tres es una herramienta matemática que se utiliza para resolver problemas de proporcionalidad. Permite encontrar un valor desconocido en una proporción cuando se conocen tres de sus valores.

Regla de Tres Directa

Si tienes una proporción del tipo:

ab=cd\frac{a}{b} = \frac{c}{d}ba​=dc​

Donde conoces aaa, bbb, y ccc, puedes encontrar el valor desconocido ddd utilizando la fórmula:

d=b×cad = \frac{b \times c}{a}d=ab×c​

Ejemplo Práctico

Supongamos que 5 manzanas cuestan 10 dólares. Queremos saber cuántos dólares costarán 8 manzanas.

  • Manzanas: 5 → 10 dólares
  • Manzanas: 8 → ¿X dólares?

510=8X\frac{5}{10} = \frac{8}{X}105​=X8​

Para encontrar X:

X=10×85=16 doˊlaresX = \frac{10 \times 8}{5} = 16 \text{ dólares}X=510×8​=16 doˊlares

Código en Java

java
public class ReglaDeTres {
    // Método que aplica la regla de tres directa
    public static double reglaDeTresDirecta(double a, double b, double c) {
        // Calcula el valor de d en la proporción a/b = c/d
        return (b * c) / a;
    }

    public static void main(String[] args) {
        double manzanas = 5;
        double costo = 10;
        double nuevasManzanas = 8;

        double nuevoCosto = reglaDeTresDirecta(manzanas, costo, nuevasManzanas);
        System.out.println("El costo de " + nuevasManzanas + " manzanas es: " + nuevoCosto + " dólares");
    }
}

Código en Go

go
package main

import "fmt"

// Función que aplica la regla de tres directa
func reglaDeTresDirecta(a, b, c float64) float64 {
    // Calcula el valor de d en la proporción a/b = c/d
    return (b * c) / a
}

func main() {
    manzanas := 5.0
    costo := 10.0
    nuevasManzanas := 8.0

    nuevoCosto := reglaDeTresDirecta(manzanas, costo, nuevasManzanas)
    fmt.Printf("El costo de %.2f manzanas es: %.2f dólares\n", nuevasManzanas, nuevoCosto)
}

Código en Python

python
def regla_de_tres_directa(a, b, c):
    # Calcula el valor de d en la proporción a/b = c/d
    return (b * c) / a

manzanas = 5
costo = 10
nuevas_manzanas = 8

nuevo_costo = regla_de_tres_directa(manzanas, costo, nuevas_manzanas)
print(f"El costo de {nuevas_manzanas} manzanas es: {nuevo_costo} dólares")

Explicación del Código

  1. Funciones/Procedimientos: En cada lenguaje, se define una función o método llamado reglaDeTresDirecta (en Python regla_de_tres_directa) que toma tres parámetros aaa, bbb, y ccc, que representan los valores conocidos en la proporción. La función calcula el valor ddd aplicando la regla de tres directa.

  2. Parámetros:

    • a: El valor que está en el numerador de la primera fracción (por ejemplo, 5 manzanas).
    • b: El valor que está en el denominador de la primera fracción (por ejemplo, 10 dólares).
    • c: El valor que está en el numerador de la segunda fracción (por ejemplo, 8 manzanas).

  3. Fórmula: La fórmula (b * c) / a se utiliza en cada lenguaje para calcular el valor desconocido ddd (en este caso, el costo de 8 manzanas).

  4. Impresión del Resultado: Finalmente, el programa imprime el resultado con un mensaje descriptivo. El formato de impresión varía ligeramente entre lenguajes, pero el concepto es el mismo.

Esta implementación de la regla de tres es básica pero suficiente para resolver problemas comunes de proporcionalidad.
