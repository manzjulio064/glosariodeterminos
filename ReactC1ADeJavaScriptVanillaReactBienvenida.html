Bienvenida

¡Les damos la bienvenida a la materia Front End III!

El desarrollo Front End abarca el conjunto de tecnologías que se emplean para desarrollar los componentes de un sitio web con los que interactúan las personas usuarias. Por eso, se suele decir que están del lado del cliente. En esta tercera materia del track de Front End, nos acercaremos a las herramientas y habilidades necesarias para trabajar con React y así crear interfaces a través de componentes reutilizables.

Nuestros objetivos son:

  • Comprender las ventajas de usar React en proyectos futuros y su correcta implementación.
  • Aprender a trabajar con componentes y conocer sus principales características.
  • Hacer uso de componentes de clase y funcionales para conocer las diferentes posibilidades que podemos llegar a encontrar en nuestra carrera profesional.
  • Utilizar correctamente los formularios con las características que React provee.
  • Incorporar el uso de hooks a los proyectos.
