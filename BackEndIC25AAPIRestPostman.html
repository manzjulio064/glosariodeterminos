Postman

Postman está disponible para usar en la web o en su versión de escritorio para Mac, Windows y Linux.

Ustedes ya lo utilizaron en otras materias, pero si aún quedan dudas o no pudieron descargarlo pueden seguir estos pasos:

Pueden descargar la última versión desde Download Postman | Try Postman for Free y elegir la versión de su sistema operativo.

Una vez descargada:

  • Linux
  • Descargar e instalar desde la consola utilizando snap con snap install postman o instalarlo desde el gestor de aplicaciones.
  • Windows
  • Luego de descargar el archivo, hacer doble clic en el archivo .exe para instalarlo.
  • Mac
  • Extraer el zip descargado y hacer doble clic en el archivo Postman. Cuando sea solicitado, mover el archivo a la carpeta “Aplicaciones”. Esto asegurará que más adelante las actualizaciones se ejecutarán correctamente.
