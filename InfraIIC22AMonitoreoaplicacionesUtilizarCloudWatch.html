Monitorear aplicaciones con CloudWatch

Ya vimos cómo usar CloudWatch para monitorear infraestructura. Exploramos el entorno de trabajo y algunas de sus métricas: CPU, memoria, espacio de almacenamiento utilizado y tráfico de red. Pero cuando nos proponemos monitorear una aplicación, estas métricas no nos resultan muy útiles.

¡CloudWatch tiene mucho para ofrecernos! Vamos a utilizar la misma herramienta para hacer el monitoreo de una aplicación web. ¡Subite a este paso a paso para hacer el despliegue de una aplicación demo utilizando Elastic Beanstalk!

¡A practicar!

¿Monitoreo e integraciones? En la siguiente ejercitación vamos a descubrir que incorporar al monitoreo otras herramientas nos dará mayor valor agregado y más productividad. ¡Y nos va a permitir recibir notificaciones en tiempo real!

Para poder hacerlo, vamos a encadenar CloudWatch, Parameter Store, Lambda y Slack para ser notificados en tiempo real mediante un pop-up en un canal de Slack cuando el estado de una instancia EC2 cambie a un valor definido como crítico.

¿Cómo lo haremos? ¡Seguí este paso a paso y olvidate de las notificaciones a través del correo electrónico!
