La clase Object

Todas las clases que creamos en Java derivan de la clase Object, aunque no esté escrito explícitamente. Por eso, cuando creamos una clase nueva, tiene ciertos métodos que hereda. De estos métodos vamos a tomar algunos y para que funcionen correctamente, debemos sobreescribirlos. El comportamiento que tienen por defecto puede causar errores o no ser el más adecuado.

Métodos heredados

  • String toString()
  • boolean equals(Object o)
  • int hashCode()
