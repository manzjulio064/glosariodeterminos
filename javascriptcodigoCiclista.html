Consigna

Sebastián es un ciclista aficionado. Su siclocomputadora (un pequeño dispositivo que lleva en su bicicleta) realiza un seguimiento de varias métricas, como la velocidad actual, la distancia, el tiempo de vuelta, la velocidad promedio de vuelta.

Dado un parámetro lapDistanceInKm, un número que interpreta la distancia en kilómetros (es decir. 1 km, 4 km, 5 km) de una vuelta, y dado otro parámetro avgSpeedByLap, una matriz de dobles, cada valor representa un promedio de velocidad en esa distancia (vuelta).

Calcular el promedio de velocidad de los 40 kilómetros más rápidos e imprimir en la salida estándar. Si la cantidad de valores no es suficiente para sus estimaciones, lance una excepción de tiempo de ejecución o un error con el siguiente mensaje: "no se alcanzo la distancia mínima en esta sesión".

Ejemplo 1:

  • lapDistanceInKm: 10
  • avgSpeedByLap: [25.7, 28.1, 29.1, 36.3, 42.4, 34.4]
  • Salida esperada: 35.55

Código Propuesto

javascript
function calcularPromedioVelocidad(lapDistanceInKm, avgSpeedByLap) {
  const distanciaTotal = lapDistanceInKm * avgSpeedByLap.length;

  if (distanciaTotal < 40) {
    throw new Error("no se alcanzo la distancia mínima en esta sesión");
  }

  // Ordenar las velocidades en orden descendente
  avgSpeedByLap.sort((a, b) => b - a);

  let distanciaAcumulada = 0;
  let velocidadAcumulada = 0;
  let vueltasSeleccionadas = 0;

  for (let i = 0; i < avgSpeedByLap.length; i++) {
    distanciaAcumulada += lapDistanceInKm;
    velocidadAcumulada += avgSpeedByLap[i];
    vueltasSeleccionadas++;

    if (distanciaAcumulada >= 40) {
      break;
    }
  }

  const promedioVelocidad = velocidadAcumulada / vueltasSeleccionadas;
  console.log(promedioVelocidad.toFixed(2));
}

// Ejemplo de uso
const lapDistanceInKm = 10;
const avgSpeedByLap = [25.7, 28.1, 29.1, 36.3, 42.4, 34.4]; // 35.55
// const avgSpeedByLap = [38.8, 38.9, 39.1, 38.8, 38.7, 38.6]; // 38.90
calcularPromedioVelocidad(lapDistanceInKm, avgSpeedByLap);
