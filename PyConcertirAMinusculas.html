Para convertir una palabra en minúsculas en Python, puedes usar el método lower(). Este método se aplica a cadenas y devuelve una nueva cadena con todos los caracteres convertidos a minúsculas.

Aquí tienes un ejemplo:

python
# Palabra original
 palabra = "HOLA"

 # Convertir a minúsculas
 palabra_minusculas = palabra.lower()

 print(palabra_minusculas)  # salida: hola

En este ejemplo, la palabra "HOLA" se convierte en "hola" utilizando el método lower().
