¡Terminamos esta clase virtual!


A lo largo de esta clase aprendimos los diferentes paquetes que nos ofrece Go. El paquete fmt para manipular el formato de las entradas y salidas estándar; el paquete os que nos brinda una interfaz para utilizar funcionalidades del sistema operativo; y, finalmente, el paquete io que nos brinda interfaces para manipular las operaciones básicas de entrada y salida. ¡A continuar aprendiendo!
