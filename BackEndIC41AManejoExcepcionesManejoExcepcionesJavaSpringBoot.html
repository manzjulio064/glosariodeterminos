Excepciones con Spring Boot

Manejar excepciones y errores en las API y enviar la respuesta adecuada al cliente es necesario para las aplicaciones empresariales. Por eso debemos aprender cómo manejar excepciones con Spring Boot que, como ya hemos visto, nos facilita el desarrollo de nuestras aplicaciones.

Spring boot nos ofrece varias alternativas a la hora de manejar los diferentes errores, vemos en detalle cuáles son.
