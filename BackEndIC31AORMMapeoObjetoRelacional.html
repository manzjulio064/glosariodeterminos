Mapeo Objeto-Relacional (ORM)

Entonces, podemos mencionar que el ORM es un mecanismo que permite interactuar con nuestra base de datos sin la necesidad de conocer SQL. Los ORMs se encargan de traducir nuestra instrucción en el lenguaje de programación que estemos utilizando a una sentencia SQL que el gestor de base de datos pueda entender.

Hay muchos ORM para Java: EJB, JDO, JPA. Mientras que estos son especificaciones, Hibernate, es una implementación.

¡No te preocupes! Algunos conceptos puede resultarte desconocidos, pero los veremos más adelante.
