Swagger

Las APIs nos permiten conectar y compartir datos entre diferentes sistemas, pero.. si queremos conectarnos con APIs externas, ¿cómo sabemos qué datos necesita recibir, qué tipo de petición enviar o a qué dirección? De alguna forma tenemos que documentar cómo usar las APIs que desarrollamos, si solo nosotros sabemos cómo conectarnos, no tendría sentido crear una API. Swagger nos ayudará a crear la documentación de nuestras APIs, para que todo el mundo entienda cómo usarla.







Pero… ¿qué es Swagger?

Swagger es un conjunto de herramientas de código abierto creadas en torno a la especificación OpenAPI, que nos ayudarán a diseñar, construir, documentar y consumir API REST.

¿Qué es Open Api?

OpenAPI es un estándar para la descripción de APIs. La especificación open API define el formato en el que se deberán describir, desarrollar, probar y documentar. Es decir, gracias a este estándar, la documentación de la mayoría de las APIs tendrá el mismo formato.
