Swagger con Spring Boot

¿Cómo funciona la documentación de APIs con Swagger y Spring Boot?

Swagger genera la documentación automáticamente a partir de nuestros controllers, creará una interfaz web en donde se mostrarán todos los endpoints de nuestra aplicación, los datos que reciben y también permitirá enviar request desde esa misma pantalla a nuestra aplicación. En nuestro caso, vamos a utilizar Swagger mediante Spring Boot, utilizando una librería llamada SpringDoc. Si querés conocer más sobre esta, te dejamos la documentación oficial.
