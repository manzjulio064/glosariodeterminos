Despliegue de APIs en Docker

Como sabemos, para crear un ejecutable de una aplicación Java, solo necesitamos ejecutar maven install, y nos creará un archivo .jar que se podrá hacer funcionar en cualquier ordenador que tenga Java instalado. Sin embargo, esto es ineficiente, ya que puede traernos varios problemas de incompatibilidad. Por lo tanto, si utilizamos Docker, tenemos la posibilidad de abstraernos de los problemas al hacer correr las aplicaciones en diferentes computadoras, veamos cómo podemos construir las imágenes y ejecutar los contenedores.







Te invitamos a ver el video para conocer más de Docker y ver cómo podemos construir las imágenes y ejecutar los contenedores.
